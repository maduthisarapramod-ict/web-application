 Student Record Management System
A simple command-line application built using Java for managing student records as a Data Structures and Algorithms practical project[cite: 6, 8].

 Features
* Register New Students[cite: 6, 7]
* View All Registered Students[cite: 6, 7]
* Filter Top Performers (Dean's List)[cite: 6, 7]
* Track Recently Added Student[cite: 6, 7]
 Data Structures Used
* **ArrayList**: To store all registered students[cite: 7, 8].
* **LinkedList**: To maintain the Dean's List (GPA >= 3.5)[cite: 7, 8].
* **Stack**: To keep a history of the last added student[cite: 7, 8].
 How to Run
1. Compile the Java files:
     bash
   javac Main.java Student.java
